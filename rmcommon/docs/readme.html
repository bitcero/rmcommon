Common Utilities 2.2
The most exciting version created so far.

Thanks for download Common Utilities 2.2. This version contains a lot of improvements over the past releases. I'm sure that you will be pleasantly surprised.

Common Utilities offers a new GUI for XOOPS control panel that improves the usability and the user experience and integrate in a better way the appearance of modules developed with Common Utilities.

Modules and plugins created with Common Utilities will be integrated in a major level with the entire system and with other modules, making the old practices obsoletes.

Important Notice

From this version and on, all updates for Common Utilities and its modules and plugins will be distributed trough automatic updates system. Full packages continue to be published in GitHub.

Novedades on this version

In this new version I've included importatn improvements in performance and functionallity of entire module:

  1. Bootstrap 3 integration.
  2. FontAwesome 4.
  3. New system of privileges and specific permissions, in addition to existing security by groups feature of XOOPS.
  4. MVC integration.
  5. Basic AR implementation.
  6. New rewriting system for URLs (only modules with support).
  7. New confgiuration system "in situ".
  8. Improved blocks manager.
  9. Improved modules manager.
  10. New groups manager with specific permissions integrated.
  11. AJAX subsystem for supported modules.
  12. Improved Admin GUI with Two·P6 theme.
  13. even much more...

Requirements

  • This version has been tested using Xoops 2.5.6 and 2.5.7.
  • PHP 5.3 o superior

To install Common Utilities please read and follow carefully next instructions.

Installation

  1. Install Xoops.
  2. Go to General Settings in Xoops Control Panel.
  3. Enable options "Enable debug" by selecting "inline mode".
  4. Save settings.
  5. Extract files in rmcommon package to your hard disk.
  6. Upload the folder system, located inside the rmcommon package, to directory modules. This folder contains theme redmexico for system module.
  7. Upload rmcommon folder to your xoops modules directory.
  8. Install rmcommon as a normal module.
    Notice: In this version the preferences are stablishind automatically during installation.
  9. Done! Common Utilities has been installed.
