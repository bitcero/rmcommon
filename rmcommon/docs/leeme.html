Common Utilities 2.2
La versión mas excitante creada hasta ahora.

Gracias por descargar Common Utilities 2.2. Esta versión contiene muchas mejoras con respecto a versiones anteriores. Estoy seguro que quedarás gratamente sorprendido.

Si es la primera vez que utilizas Common Utilities, debes saber que es un framework para el desarrollo de módulos y plugins más modernos y funcionales que los módulos estándar de XOOPS. Al utilizar Common Utilities como base para el desarrollo de tus módulos, podrás tener acceso a un paradigma totalmente diferente y mas productivo que el convencional.

Además, los módulos desarrollados usando Common Utilities mantienen una mejor integración con otros módulos, plugins y el sistema completo, permitiéndote extender las capacidades de XOOPS hasta donde tu imaginación te lo permita, es decir, no hay límites para lo que puedes hacer.

Aviso Importante

A partir de la versión RC todas las actualizaciones para Common Utilities, sus módulos y plugins, se distribuyen a través del sistema de actualizaciones automáticas. Las versiones completas seguirán publicándose en GitHub.

Novedades en la versión 2.2

En esta nueva versión he incluido muchas mejoras, incluyéndo las siguientes:

  1. Integración de Bootstrap 3.
  2. Integración de FontAwesome 4.
  3. Nuevo sistema de control de privilegios y permisos específicos, adicional al sistema existente.
  4. Integración completa del sistema MVC para los módulos nuevos.
  5. Implementación parcial de AR.
  6. Nuevo sistema de reescritura de URLs (para módulos que lo soporten).
  7. Nuevo sistema de configuración in situ.
  8. Administrador de bloques mejorado.
  9. Administrador de módulos mejorado.
  10. Nuevo administrador de grupos.
  11. Subsistema AJAX para operaciones con módulos que lo soporten.
  12. Interfaz gráfica mejorada mediante el tema Two·P6.
  13. Y mucho más...

Requerimientos

  • Esta versión ha sido probada en Xoops 2.5.6 y Xoops 2.5.7.
  • PHP 5.3 o superior

Para instalar Common Utilities debes leer y seguir cuidadosamente las siguientes instrucciones.

Instalación

  1. Instala XOOPS.
  2. Ve al panel de control -> preferencias -> preferencias generales.
  3. Activa la opción "Activar modo Debug" seleccionando la opcion "modo
    en linea".
  4. Guarda las preferencias.
  5. Extrae los archivos del paquete a tu disco duro.
  6. Sube la carpeta system ubicada en el paquete de Common Utilities al directorio /modules/. Esta carpeta contiene el tema redmexico para el módulo system.
  7. Sube la carpeta rmcommon al directorio modules de xoops.
  8. Instala rmcommon como un módulo normal.
    Nota: En esta versión de Common Utilities las preferencias son establecidas automáticamente.
  9. Listo. Common Utilities se ha instalado.

Obtener Common Utilities

Puedes obtener nuevas versiones y actualizaciones de Common Utilities directamente en el repositorio en GitHub. Tambien puedes hacer un fork de nuestro repo.
