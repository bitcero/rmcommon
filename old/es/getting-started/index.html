Requisitos previos

Antes de instalar Common Utilities, es necesario contar con algunos sencillos requisitos para que todo funciones correctamente.

PHP 5.4

El módulo ha sido desarrollado utilizando PHP 5.4 y constantemente se integran las nuevas características del lenguaje.

Instalación

Instalar Common Utilities es muy sencillo. Estas son las instrucciones.

La instalación de Common Utilities es similar a la de cualqueir otro módulo en XOOPS, sin embargo es necesario poner especial atención a ciertos detalles para asegurarnos que todo marche correctamente.

1. Descomprimir archivo

Una vez descargado el archivo que contiene los archivos el módulo, es necesario extraer el contenido al disco duro local. Una vez extraídos, podrás observar los siguientes archivos:

  • rmcommon*
  • .gitattributes
  • .README.md

El subdirectorio system contiene el tema para el módulo sistema que activa la interfaz gráfica de Common Utilities.

2. Subir archivos

La carga de archivos al servidor donde se encuentra instalado XOOPS se realiza en dos partes:

1. Carga del tema para el módulo System

Common Utilities proporciona un tema para ser integrado en el módulo de sistema. Este tema proporciona los medios para poder modificar la interfaz gráfica y utilizar los componentes visuales nuevos.

2. Carga del módulo

Por supuesto es necesario cargar los archivo propios de Common Utilities, que se integrarán con XOOPS y ofrecerán nuevas capacidades y elementos listos para ser usados.

Common Utilities no modifica ningún archivo de XOOPS, siguiente la filosofía 0% hacks ➙ 100% funcionalidad.
Importante: para la carga de archivos a un servidor remoto, es probable que necesites utilizar algún cliente FTP cómo FileZilla o algún programa similar.

Para subir los archivos a tu sitio XOOPS, sigue el procedimiento a continuación:

  1. Sube la carpeta system ubicada dentro de la carpeta rmcommon al directorio /modules/system/themes de tu instalación XOOPS.
  2. Sube el directorio rmcommon al directorio /modulesde tu instalación XOOPS.

Una vez los archivos se encuentren alojados en el servidor, procederemos a instalar Common Utilities, para lo cuál seguiremos el procedimiento básico de instalación de módulos en XOOPS.

3. Instalar el Módulo

  1. Ingresa al panel de control de XOOPS, posteriormente ve al administrador de módulos de XOOPS.
  2. Haz click en el botón Instalar módulos (Install module). Podrás ver el icono de Common Utilities en la lista de módulos disponibles para instalación.
  3. Haz click en el icono Instalar (representado por una flecha), Nuevamente en el botón Instalar.
  4. Si todo ha resultado correctamente, Common Utilities está instalado y listo para ser utilizado.

Probablemente te preguntes acerca de la instalación del tema del módulo System y otros ajustes; no te preocupes, Common Utilities ha preparado todo el sistema para funcionar adecuadamente.
