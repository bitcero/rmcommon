Requirements

Before installing Common Utilities, you need to have some simple requirements for all functions properly.

PHP 5.4

The module has been developed using PHP 5.4 and, the new language features are constantly integrated.

XOOPS 2.5

Common Utilities is a XOOPS module, so, before to install it, it is necessary to have installed and configured XOOPS.

Get XOOPS

Installation

Install Common Utilities is very easy. Here are the instructions.

Common Utilities installation is similar to any other module in XOOPS, however you need to pay special attention to certain details to make sure everything runs properly.

1. Unzip file

Once you download the compressed file containing the module files, you must extract the contents to local hard disk. Once extracted, you will see the following folder and files:

  • rmcommon*
  • .gitattributes
  • .README.md

The subdirectory system contains the theme for system module. This theme activate the GUI from Common Utilities.

2. Upload Files

Uploading files to the server whee is installed XOOPS is in two partes:

1. Uploading theme to the system module

Common Utilities provides a theme to be integrated with system module. This theme provides all the GUI functionallity.

2. UPload the module

Of course you need to upload the Common Utilites files, which will be integrated with XOOPS and provide new features and ready to use elements.

Common Utilities does not modify any files in XOOPS, following the philosophy 0% hacks ➙ 100% functionality.
Important: for uploading files to a remote server, you will probabily need to use a FTP client such as FileZilla or similar program.

To upload files to your XOOPS site, follow the procedure below:

  1. Upload the folder system located into folder rmcommon to directory /modules/system/themes of your XOOPS site.
  2. Upload folder rmcommon to directory /modules of your XOOPS installation.

Once the files are hosted on the server, we will install Common Utilites, for which we follow the basic procedure for installing modules in XOOPS.

3. Install the Module

  1. Login to the XOOPS control panel, then go to the XOOPS modules manager.
  2. Click on the button Install Module. You will se the icon of Common Utilities from the list of available modules of installation.
  3. Click on the icon Install (represented by an arrow), then click on the next button Install.
  4. If everything has been correctly Common Utilities is installed and ready for use.

You are probably wondering about the activation of the theme for System module and other settings; do not worry, Common Utilities has prepared the entire system to function properly.
