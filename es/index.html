Un framework

Todas las herramientas para crear nuevos módulos: más modernos, más funcionales y mejor integrados.

Nueva apariencia

No solo inyecta potencia a XOOPS, también embellece la apariencia general del panel de control.

Compatibilidad

Aprovecha las ventajas que ofrecen los nuevos módulos, sin sacrificar compatibilidad con los anteriores.

La mejor base para módulos XOOPS nuevos

Common Utilities ofrece una excelente base para la creación de nuevos módulos, gracias a sus componentes mejorados. Crea módulos que se integran totalmente entre ellos y con el sistema con una curva de aprendizaje muy ligera.

Si desarrollas módulos para XOOPS, ya has recorrido el 90% del camino.

Módulos más útiles y con mejor apariencia

Common Utilities no genera incompatibilidades con módulos existentes de XOOPS, por lo que puedes seguir utilizándolos sin problemas, pero si deseas mejorar tu sitio web, cuenta con excelentes módulos 100% nativos que te permitirán crear blogs, páginas, portafolios, temas, etc..

Actualizaciones 1-Click

Mantén actualizados tus módulos fácilmente. Olvídate de buscar actualizaciones manualmente y todo el proceso que eso conlleva; deja que Common Utilities haga todo el trabajo por tí mediante su sistema de actualizaciones 1-Click*.

Si eres desarrollador, prepara tus módulos para aprovechar el sistema de actualizaciones integrado; es muy sencillo.

* Disponible únicamente para módulos nativos de Common Utilities.

Características
Common Utilities ofrece características que permiten una mejor experiencia para los usuarios finales y mejores herramientas para desarrolladores.

Administrador de Módulos

Un nuevo administrador de módulos; más facil de usar y entender.

Bloques Mejorados

Common Utilities ofrece su propio sistema de bloques que permiten crear tantas posiciones como sean necesarias y un manejo mucho mas intuitivo y funcional de los bloques.

Seguridad Mejorada

Una nueva capa de seguridad que permite establecer permisos por grupos y acciones específicas que además pueden integrarse fácilmente en otros módulos.

Manejo de Usuarios y Grupos

Un administrador de usuarios y grupos que permite aprovechar las nuevas características de seguridad de Common Utilities.

Administrador de Imágenes

Un administrador de imágenes muy versátil que permite un manejo más intuitivo de las imágenes y que puede ser integrado en cualquier módulo.

Soporte para Plugins

Es posible agregar mayor funcionalida o extender las características existentes a través de plugins.

Sistema de Comentarios

Un nuevo sistema de comentarios, más fácil de integrar y de usar con cualquier módulo.

Actualizaciones 1-Click

Mantén tu sistema al día y recibe las actualizaciones directamente con Common Utilities con un solo click.

Sistema de Configuración

Con el sistema de configuración integrado es posible establecer las opciones para los módulos sin abandonar lo que estas haciendo. Un nuevo método en línea permite configurar módulos vía AJAX.

Documentación en Línea

Un método de documentación que permite a los módulos, plugins y temas proporcionar documentación alojada en Internet fácilmente.

Nuevo Control de Plantillas

Añade una nueva clase para plantillas que permite un manejo mas efectivo y sencillo de hojas de estilo, scripts y otros elementos.

Nuevos Editores de Texto

Elige entre los distintos editores que ofrece Common Utilities de acuerdo a tus gustos de edición: Visual, HTML, XoopsCode o Simple.

Soporte Multiidioma

Un sistema de localización totalmente independiente de XOOPS, basado en GetText y con archivos fácilmente editables con PoEdit. y disponible para todos los módulos, incluso módulos antiguos.

Códigos Personalizados

Soporte para código personalizados que pueden ser utilizados en módulos, plugins o temas y que pueden extender la funcionalidad por defecto del sistema.

Implementación MVC

Implementación básica del patrón MVC aplicado a módulos XOOPS que permite un desarrollo organizado, estructurado y en tiempo récord.

Campos de Formulario

Campos de formulario adicionales a los existentes en XOOPS, mejor integrados y mucho más sencillos de usar.

Reescritura de URLs

Permite un fácil control de reescritura de las URLs por parte de los módulos, y para los módulos que implementan MVC la reescritura ya está integrada.

Soporte de temas

La apariencia general puede ser modificada mediante temas para el panel de control.

¡Transforma XOOPS por dentro y por fuera!

XOOPS y Common Utilities son la pareja perfecta para tus proyectos web.

Módulos Disponibles

Esta es una lista de los módulos disponibles para Common Utilities, pero recuerda que siempre puedes utilizar los módulos convencionales de XOOPS sin problemas de compatibilidad.

  • MyWords

    Un módulo para publicar artículos o noticias con todas las herramientas para crear una sección de blog.

  • QuickPages

    Módulo para la publicación de páginas semi-estáticas con avanzadas características para el uso de plantillas.

  • xThemes

    Framework para la creación de temas avanzados para XOOPS. Proporciona nuevas posibilidades para crear temas profesionales.

  • Professional Works

    Módulo para crear una sección de portafolio profesional. Ideal para creativos, agencias publicitarias, freelancers, etc.

  • bXPress

    Módulo ligero para la creación de foros en XOOPS. Caracteristicas novedosas frente a otras opciones.
