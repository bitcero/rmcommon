Módulos

Módulos disponibles nativos de Common Utilities.

Importante

La instalación de Common Utilities en XOOPS no impide, ni obstaculiza, el uso de otros módulos. Podrás seguir utilizando los mismo módulos que has usado hasta el momento, si así lo deseas, mientras usas los nuevos módulos para Common Utilities.

MyWords

Es un módulo ideal para crear blogs con XOOPS. Sencillo de utilizar y con características únicas, muy distintas a los módulos existentes.

xThemes

Un framework muy completo para el diseño y desarrollo de temas con calidad premium. xThemes brinda a los temas XOOPS el poder de los módulos, convirtiéndolos en verdaderos componentes integrados con XOOPS.

QuickPages

Un módulo para la publicación de páginas y contenido en XOOPS. Permite crear diversos tipos de páginas para mercadotecnia, como páginas de atterizaje, de ventas, de agradecimiento, páginas de productos, informativas, etc.
